br and hr tags

The br tag acts as a line break and will make the next html element go to the next line

This text tag in two lines down now since we have two br tags above it

The hr tag is a br tag with an added line to help break up content

You may have noticed that the p tag already places itself on the next line. That is because it a block level element.

With HTML there are block level and inline level elements. Block level means that the element will automatically get placed on the next line where as inline elements will try to stay on the same line as long as there is space to do so. If there isn't enough space to fit that element it will drop to the next line.

Links

Links are how you go from one page to another.

click me to go to google

something new to notice in the a tag is the href.

some tags have something that's called an attribute and attribute allows us to add extra functionality to the tag

another attribute that we can add to the a tag is called target

if we assign a target attribute with a _blank value this will cause the link to open in a new tab as opposed to the same page.

this is generally used when you want to show a user a different page but you don't want them to leave your site altogether.
